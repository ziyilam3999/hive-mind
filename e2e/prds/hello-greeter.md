# PRD: Hello Greeter

## Problem Statement
Need a single TypeScript function that greets users by name.

## Requirements
- REQ-01: Export a `greet(name: string): string` function from `src/greeter.ts`
- REQ-02: Returns `"Hello, {name}!"`

## Success Criteria
- `greet("World")` returns `"Hello, World!"`

## Out of Scope
- No CLI, no tests beyond the success criterion, no dependencies
